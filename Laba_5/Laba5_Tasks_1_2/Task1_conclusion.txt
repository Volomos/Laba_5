При работе с коллекциями на 10тыс. элементов получены такие результаты: 
Создание коллекций заняло ~одинаковое время, кроме Tree Set (дольше).
Работа с первым и последним элементами была лучше в Array List.
В Linked List наблюдался самый длительный поиск первого элемента, хотя поиск элемента в средине коллекции произвелся в Linked List быстрее всех других. 
Для нахождения нужного элемента в Linked List перебирается не больше половины списка, но с точки зрения асимптотического анализа время на поиск растет линейно — O(n). 
При работе с коллекциями на 1000тыс. элементов Hash Set работал быстрее остальных коллекций.
